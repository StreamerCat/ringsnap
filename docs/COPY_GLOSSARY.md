# RingSnap Copy Glossary

## Core Terminology Replacements

| Original / Forbidden Term | Preferred Replacement | Context / Notes |
| :--- | :--- | :--- |
| **AI** | **Agent** | "RingSnap Agent", "Your Agent" |
| **AI Receptionist** | **Virtual Receptionist** | Also "Digital Receptionist" or just "Receptionist" |
| **AI Assistant** | **Voice Agent** | "Assistant" is okay, but "Voice Agent" is more precise |
| **AI Phone Number** | **RingSnap Number** | Or "Business Line", "Dedicated Number" |
| **Bot / Chatbot** | **System / Assistant** | Never use "Bot". It implies low quality. |
| **Artificial Intelligence** | **Automation / Technology** | Use "Advanced Automation" if needed. |
| **Train / Training** | **Configure / Setup** | "Configuring your agent" feels more controllable. |
| **Hallucination** | **Error / Mistake** | Internal or support contexts only. |

## Voice & Tone

*   **Benefit-First**: Don't say "We use LLMs". Say "We answer calls 24/7 so you don't miss jobs."
*   **Contractor-Centric**: Use terms like "booked jobs", "missed calls", "crews", "dispatch".
*   **Professional & Reliable**: The agent isn't a "cool tech toy"; it's a "reliable employee".

## Safe "AI" Usage (Exceptions)
*   **Privacy Policy / TOS**: Where legal distinction between human and automated processing is required.
*   **Technical Docs**: If explaining the underlying STT/LLM/TTS stack to developers (rare).
*   **SEO Hidden Text**: Only if strictly necessary for high-volume keywords, but prefer user-facing replacement.
