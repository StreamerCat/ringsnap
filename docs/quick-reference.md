## Call Forwarding Codes

### Enable Forwarding

| Carrier | Code |
|---------|------|
| AT&T | `*21*[RingSnap#]*` |
| Verizon | `*72` + [RingSnap#] |
| T-Mobile | `**21*[RingSnap#]#` |
| Sprint | `*72` + [RingSnap#] |
| Landline | `*72` + [RingSnap#] |

### Disable Forwarding

| Carrier | Code |
|---------|------|
| AT&T | `#21#` |
| Verizon | `*73` |
| T-Mobile | `##21#` |
| Sprint | `*720` |
| Landline | `*73` |

## Common Troubleshooting

| Issue | Quick Fix |
|-------|-----------|
| Calls not forwarding | Re-enter forwarding code, test from another phone |
| Can't log in | Use "Forgot Password", check email, clear cache |
| No calls in dashboard | Verify forwarding is active, wait 2-3 minutes for sync |
| AI not following instructions | Save instructions, changes apply to next call |
| Recording not working | Verify Pro/Premium plan, enable in Settings |
