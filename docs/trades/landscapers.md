## Recommended Custom Instructions

Copy and customize for your landscaping business:

```
Always ask if this is for residential or commercial property.
For new customers, ask approximate property size (sq ft or lot size).
Mention we offer weekly, bi-weekly, and monthly maintenance plans.
Ask if they have an irrigation system (helps with service planning).
We're licensed, insured, and offer free estimates for new projects.
Our service area is [your area].
We service properties of all sizes from small yards to large estates.
```

### Seasonal Services

```
Based on season, mention available services:

Spring: Cleanup, mulching, aerating, planting
Summer: Mowing, trimming, irrigation, pest control
Fall: Leaf removal, winterizing, final cleanup
Winter: Snow removal (if offered), holiday lighting

Ask if they're interested in seasonal packages.
```

## Property Sizing Questions

Help qualify leads accurately:

```
For accurate quotes, ask:
- Approximate yard size (small, medium, large, or sq ft)
- Front yard, back yard, or both?
- Is there a lot of landscaping/beds or mostly lawn?
- Any hills or slopes?
- Fenced areas or access issues?

This helps us prepare an accurate estimate.
```

## Emergency Handling

### Emergency Policy Example

```
For tree emergencies (fallen tree, blocking driveway, on structure):
1. Ask if anyone is hurt or in danger
2. If blocking driveway or on house, treat as urgent
3. Transfer to my cell: (303) 555-1234

For irrigation leaks/flooding:
- Ask them to shut off system if possible
- Book same-day if water is actively running
- Otherwise next available

For storm damage cleanup:
- Book within 48-72 hours based on severity
```

## Commercial vs Residential

Different approaches for each:

### Residential Calls

```
For residential customers:
- Schedule free estimate at their convenience
- Ask about homeowner goals (curb appeal, low maintenance, etc.)
- Mention neighbor referral discounts
```

### Commercial Calls

```
For commercial properties:
- Ask for property management company name if applicable
- Get property address and size
- Schedule site walk-through
- Mention our commercial contracts and reliability
- Ask about multi-property discounts
```

## Pricing Questions

Landscaping pricing varies significantly:

### What to Tell Your AI

```
For pricing questions, we provide free estimates because every property differs.

General ranges we can share:
- Basic lawn mowing: $30-75 per visit depending on size
- Full yard cleanup: $150-500 depending on size and condition
- Mulching: $40-100 per yard of mulch plus labor
- Landscape design/install: Varies widely, need consultation
- Irrigation repair: $75-200 for most repairs

We always provide a free written estimate before any work.
```
